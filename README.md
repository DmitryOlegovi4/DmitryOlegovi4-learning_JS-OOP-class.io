Необходимо реализовать класс, который описывает статью, у класса должны быть реализованы свойства:
- заголовок
- текст
- кол-во лайков
(заголовок и текст должны быть с геттерами и сеттерами)

должен быть реализован метод:
- добавить лайк посту
